Run locally (required)

Do not open this file as file:// — browsers block cross-origin requests. Serve this folder over HTTP.

  1. Terminal 1 — API: cd backend && uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
  2. Terminal 2 — static server: cd demo && python3 -m http.server 8080
  3. Browser: open http://127.0.0.1:8080 (or http://localhost:8080)

Remote API: append ?api=https://your-backend.example.com to the URL.
