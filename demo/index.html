Quick start

  • 1 Connect to the backend API
  • 2 Seed a demo provider passport
  • 3 Run a credentialing workflow

Run locally ▾

  1. Terminal 1 — API: cd backend && source .venv/bin/activate && uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
  2. Terminal 2 — demo: cd demo && python3 -m http.server 8080
  3. Browser: open http://127.0.0.1:8080

Remote API: append ?api=https://your-backend.example.com to the demo URL.
